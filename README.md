HKEX_Agent

The agent would receive a user prompt to inquire some aspects about one specific company across certain periods.
After analyzing the user prompt, the agent should specify key parameters, such as company name, date range.
A tool would be given to the agent for fetching HKEX filings. 
for each file, a specific prompt would be used to extract certain information. 
The agent then would collect all those extracted information and summarize.
The agent would reflect on the draft summary and revise.
The agent would give a final response to the user.
